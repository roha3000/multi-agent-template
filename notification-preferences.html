Setup Instructions:

1. Configure Twilio credentials in .env: TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_FROM_NUMBER

2. Configure SendGrid API key in .env: SENDGRID_API_KEY, SENDGRID_FROM_EMAIL

3. Restart the global context manager: npm run monitor:global

4. Enable notifications and configure your phone/email above.
